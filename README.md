### 🚀 About me

Backend &amp; AI engineer with **3+ years** shipping software for real businesses. My path started with **curiosity and cybersecurity** — Kali Linux, SQLmap, learning how systems break — and grew into **backend, integrations and AI**. Today I design and run **multi-agent LLM systems**, **RAG pipelines**, and the **APIs and data layers** that keep them reliable.

- 🤖 Shipped a **multi-agent customer-support system** to production — async FastAPI · LLM router → specialized agents + tools + human handoff · Postgres + Redis · CI/CD + agent evals
- 📄 Built a **live multimodal RAG agent** → **[pdf-rag-agent.vercel.app](https://pdf-rag-agent.vercel.app)** — reads document pages (figures &amp; diagrams included) and answers **citing the source page**
- 🔄 Real-time **Odoo ↔ marketplace** connectors (MercadoLibre, Amazon, Walmart, Shopify) and end-to-end ERP automations
- 🧠 I work in **Claude Code every day** to design, direct and build — *I direct the AI, I don't hand it the wheel*
